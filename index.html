This is a very simple repo to collect some MS Office alternatives that are both free and linux compatible.
  1. LibreOffice
    open source and support for MS Office files - available for Linux, MacOS and Windows
    most popular option, not my preferred one tho

  2. OpenOffice
    open source and support for MS Office files - available for Linux, OS X and Windows (32-bit)
    relatively abandonded, no major updates for over a decade/no support for Windows (64-bit)

  3. Calligra
    open source and hit-or-miss support for MS Office files - available for Linux
    ships with most KDE distros

  4. WPS Office
    closed source and support for MS Office files - available for Linux, MacOs and Windows
    closest UI to MS Office, has paid tiers and is my preferred option
